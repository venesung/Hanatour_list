하나투어 앱 이용을 위한 접근 권한 안내

필수 접근 권한
  • 기기ID 및 통화정보: 휴대전화 상태 및 ID 읽기
  • 네트워크 접근 : 네트워크 연결 정보 확인
선택 접근 권한
  • SMS : SMS 메시지 보내기
  • 저장: USB 저장소의 콘텐츠 읽기,
    USB 저장소의 콘텐츠 수정 또는 삭제
  • 전화: 전화번호로 직접 전화 걸기,
    휴대전화 상태 및 ID 읽기
  • 주소록 : 기기에서 계정 검색
※ '선택 접근 권한'은 해당 기능을 사용할 때 허용이 필요하며 비허용시에도 해당 기능 외 서비스 이용이 가능합니다.
접근권한 변경 방법
스마트폰 설정 > 애플리케이션 관리 또는 앱 > 하나투어에서 각 권한별 변경 가능
